# シンガポール所得税シミュレーター

シンガポール在住で海外から収入を得た場合の所得税を計算するWebアプリです。収入のタイプによる課税判定(そもそも課税されるか)から、累進ブラケットごとの税額内訳まで確認できます。

**公開ページ**: https://yoshihitoijichi.github.io/singapore-tax-simulator/

## 機能

- 収入タイプ別の課税判定
  - 海外クライアントの仕事をシンガポール国内で行う → 国内源泉所得として課税
  - シンガポール企業に雇用 → 累進課税(外国人はCPF対象外)
  - 純粋な海外源泉所得(国内での業務なし) → 個人受領は原則非課税
- 税務居住者(年183日以上)/非居住者の切替(非居住者は給与一律15%か累進の高い方、事業所得等は一律24%)
- 累進ブラケット(0〜24%)ごとの税額内訳をバーと表で表示
- 為替レート入力による日本円換算、実効税率・限界税率の表示
- 年間収入は最大S$1,000万(約11.5億円)まで入力可能
- ライト/ダークモード対応・依存ライブラリなしの単一HTMLファイル

## 計算の前提(YA2025 / 2024年所得 基準)

- 居住者税率はIRAS公表の累進税率表(0〜24%)を使用
- 控除は勤労所得控除S$1,000のみ反映した簡易計算(扶養控除・CPF関連控除等は未反映)
- YA2024の税額リベート等の一時措置は未反映
- 事業所得は経費控除後の純利益を入力する想定

## 実行方法

`index.html` をブラウザで開くだけで動作します。

## 免責

本アプリの計算結果は概算です。実際の課税判定(源泉地・居住性・租税条約の適用)は個別事情によります。正確な判断はIRASまたは税理士にご確認ください。日本側の税務(非居住者判定・国外転出時課税等)は対象外です。
